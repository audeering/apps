Diese werden an einen Server von audEERING übertragen und dort von einer Emotionserkennungssoftware analysiert. Innerhalb von Sekunden erhält der Nutzer das Ergebnis der Emotionsanalyse: die erkannte Emotion wird benannt und mit einem prozentualen Konfidenzmaß versehen, das ausdrückt, wie sicher sich die Software bei der Emotionserkennung ist.
